# jarvis bot

A GitHub App built with [Probot](https://github.com/probot/probot) that do everything we need.

- Calculate the PR size and add label to it
- Generate release note when you push tag
- Customize command
...

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```
